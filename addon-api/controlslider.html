ControlSlider-class

Derived from: public BSlider

Declared in: Controls.h

Overview

This is a class that is very similar to BControl. The only difference is that this spawns another thread to do the mouse-tracking, thus leaving the window and its views free to respond to messages (i.e. the window will be unlocked most of the time).

Constructor

ControlSlider()

  ControlSlider(BRect frame, const char* name, const char*
    label, BMessage *message, int32 rangeMin,

This does nothing except passes the parameters straight to BSlider's constructor.

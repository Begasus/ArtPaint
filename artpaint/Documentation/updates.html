Downloading Updates

New versions of ArtPaint can be downloaded from two different locations. These locations are
  1. http://www.be.com/beware/Graphics/ArtPaint.html
  2. http://www.helsinki.fi/~hsuhonen/artpaint/

The second location is ArtPaint's official homepage and it will in future contain more material such as tips on using the program and perhaps an image-gallery.

Downloading New Add-ons

New add-ons will be released for ArtPaint between actual new versions of ArtPaint. These add-ons can be obtained from the following location:
  1. http://www.helsinki.fi/~hsuhonen/artpaint/

There is also an add-on developer kit available from the above page. Developers who wish to make new add-ons for the program are welcome to download it.
